# Unity Package Template

This is a starter Unity project template pre-configured with:

- **UniTask**: Asynchronous async/await integration for Unity.
- **UniRx**: Reactive Extensions for Unity.

## Getting Started

1. Open this folder in Unity Hub.
2. Unity will automatically download the packages defined in `Packages/manifest.json`.

## Dependencies

- [UniTask](https://github.com/Cysharp/UniTask)
- [UniRx](https://github.com/neuecc/UniRx)
